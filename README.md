# 🚗 AI Car Price Predictor

An interactive Machine Learning web app that predicts car prices based on user input features. Built using **Python, Scikit-learn, and Streamlit** with an animated UI.

---

## 🔥 Features

* 🚗 Predict car price using ML model
* 🎯 Accurate regression model
* 🎮 Interactive Streamlit UI
* 🌙 Animated background (running car)
* ⚡ Real-time prediction
* 🎨 Dynamic UI with effects

---

## 📊 Input Features

* Symboling
* Car Body
* Drive Wheel
* Engine Location
* Wheelbase
* Car Length
* Car Width
* Car Height
* Engine Type
* Cylinder Number
* Engine Size
* Horsepower
* City MPG

---

## 💰 Output

* Estimated Car Price (based on ML model)

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## 🚀 How to Run Locally

### 1️⃣ Clone the repository

```bash
git clone https://github.com/subhit-coder/car-price-predictor.git
cd car-price-predictor
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the app

```bash
streamlit run app.py
```
